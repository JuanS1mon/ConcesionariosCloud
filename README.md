# ConcesionariosCloud

Proyecto para la gestión de concesionarios de autos, compuesto por backend y frontend.

## Estructura
- `backend/`: API REST (submodule).
- `frontend/`: Aplicación web (submodule).
- `docs/`: Documentación del proyecto.
- `tools/`: Herramientas auxiliares.

## Instalación
1. Clona este repositorio con submodules: `git clone --recursive https://github.com/JuanS1mon/ConcesionariosCloud.git`
2. Consulta los guías en `docs/` y `DEPLOYMENT_GUIDE.md`.
